# 联邦图学习研究发展综述

联邦图学习（Federated Graph Learning，FGL）将联邦学习的分布式协同训练思想引入到图数据建模中，旨在在保证各参与方图数据不出本地的前提下，联合训练图机器学习模型。近年来该领域发展迅速，本文从整体发展脉络、安全问题、异构图学习和推荐系统四个方面，按时间顺序梳理联邦图学习研究的演进，列举里程碑式的代表论文（包括标题、作者、年份、发表 venue 及引用次数），并分析各代表工作之间的逻辑联系。在此基础上，总结当前联邦图学习面临的挑战与未来研究趋势。

# 1. 整体发展脉络

联邦图学习作为联邦学习与图神经网络（Graph Neural Network, GNN）的交叉产物，于近年逐渐兴起。2020年前后出现了首批探索性工作：

- GraphFL: A Federated Learning Framework for Semi-Supervised Node Classification on Graphs - Binghui Wang 等人（2020年 arXiv；2022年发表于 ICDM）- 引用约171次 1 2。这被认为是最早的联邦图学习框架之一，也是首个面向图半监督节点分类的联邦学习方法 1 。GraphFL通过元学习思想缓解客户端数据非独立同分布（non-IID）问题，并设计自训练机制利用未标记数据 3 。实验表明在多个图数据集上性能优于传统联邦学习基线 4 。GraphFL开创了联邦学习在图节点分类任务上的先河。  
- Vertically Federated Graph Neural Network for Privacy-Preserving Node Classification (VFGNN) - Chaochao Chen 等人（2022年IJCAI，最初2020年提出）- 引用约30+次56。VFGNN针对垂直数据划分场景，是首个支持“纵向联邦”下图神经网络训练的框架5。不同于GraphFL假设各客户端拥有不同子图的“水平联邦”场景，VFGNN考虑图的特征和结构由不同数据方持有，提出将GNN计算图拆分为客户端和服务器两部分，服务器承担中间层计算，同时引入差分隐私防止服务器推理敏感信息5。VFGNN解决了异主体间特征和邻接关系隔离的难题，2022年正式发表后丰富了联邦图学习的应用场景。

2021年是联邦图学习方法的快速发展期，多种改进框架涌现：

- FedGraph: Federated Graph Learning with Intelligent Sampling - Fahao Chen 等人（2022年IEEE TPDS，发表于2021年）- 引用约17次 7 8 。该工作关注大规模图的训练效率，提出利用深度强化学习智能采样子图，以降低跨客户端的计算和通信开销 9 。同时，FedGraph设计了跨客户端卷积操作，避免直接分享节点特征来减轻隐私泄露 10 。实验表明相比不采样策略，FedGraph在收敛速度和精度上取得更优折中 11 。此方法是首批关注联邦图学习效率优化的研究之一。  
- Fedora: Federated Graph Learning Framework with Global Self-Supervision - Chuan Chen 等人 (2021 年 arXiv) - 引用次数未详 12。FedGL引入全局自监督信息来提升联邦图模型质量 12。服务器收集各客户端上传的预测分布和嵌入，构造全局伪标签和全局伪图并下发，从而为本地模型提供额外的伪监督信号，弥补各客户端局部图结构和标签的不足 13。这种方法有效缓解了不同客户端图数据的异质性和不完整性 14。FedGL是联邦图学习中利用自监督信号缓解数据孤岛问题的代表性工作。  
- ASFGNN: Automated Separated-Federated Graph Neural Network - Jun Zheng 等人（2021年Peer-to-Peer Networking and Applications）- 引用次数未详。该工作探索自动化和分离式的联邦图模型训练，提出将本地GNN训练与全局聚合解耦，利用神经架构搜索自动优化本地模型结构。尽管

ASFGNN使用了常规权重平均聚合，可能导致信息损失 15 16，但其创新之处在于引入AutoML理念，开启了联邦图学习模型自动化设计的方向。

2022年联邦图学习迎来了标志性成果和更广的关注：

- A Federated Graph Neural Network Framework for Privacy-Preserving Personalization (FedPerGNN) - Chuhan Wu 等人（2022年 Nature Communications）- 引用173次 17。该工作（在 arXiv题为FedGNN 18）将联邦图学习成功应用于个性化推荐场景，是领域里程碑式成果。FedPerGNN 提出在每个用户客户端本地基于用户-物品子图训练GNN模型，服务器聚合梯度更新，同时采用本地差分隐私扰动梯度保护用户数据 19 20 。另外引入伪交互项来隐藏真实交互物品 20 。该方案有效利用了用户行为图的高阶关系，在多个数据集上性能接近集中式训练且保护了用户隐私 21 。作为在顶级学术期刊发表的工作，FedPerGNN显著提升了联邦图学习在工业界和学术界的影响力。  
- Cluster-driven Graph Federated Learning over Multiple Domains - Daniele Caldarola 等人 (2021年 CVPR Workshops) - 引用次数未详。该工作面向多领域图数据，通过聚类驱动策略缓解不同域之间分布差异，以提高全局模型泛化性能 22。这是早期探索跨域联邦图学习的工作之一，与FedGL等方法一道，为应对数据异质性提供了新思路。  
- Fede: Embedding Knowledge Graphs in Federated Setting - Min Chen 等人（2021年IJCKG）-引用次数未详 23。这是一项将知识图谱嵌入引入联邦学习的尝试。FedE框架使多个机构可以在不共享各自知识图谱数据的情况下学习联合的知识表示 24。该研究首次讨论了多关系异质图的联邦表示学习问题，为异构图场景下的联邦学习奠定基础（详见下文第3节）。

2023年以来，研究重点逐步从构建框架转向解决关键挑战，涌现出一批具有高度影响力的方法：

- Federated Learning on Non-IID Graphs via Structural Knowledge Sharing (FedStar) - Yue Tan等人（2023年AAAI）- 引用约90次25 26。FedStar针对不同客户端图结构差异大的问题，提出了结构知识共享机制27。它显式提取每个客户端的结构模式（通过独立的结构编码器获得“结构嵌入”），并在全局共享该结构编码器，以捕获跨域结构共性28。同时将节点特征相关的表征保留为个性化部分，从而避免不同客户端间特征分布不一致造成的对齐困难29。通过在跨数据集、跨领域的non-IID场景下实验，FedStar展示了显著优于以往方法的性能30。这一工作在联邦图领域被视为应对数据异质性的突破，发表当年已获得大量引用。  
- FedHGN: Federated Heterogeneous Graph Neural Network for Privacy-Preserving Recommendation - Bo Yan 等人（2023年 arXiv）- 引用次数未详。该方法将联邦学习扩展到异构图推荐场景：如用户- 物品二部图包含不同类型节点和边。FedHGN设计了针对异构图结构的联邦训练框架，并结合隐私机制保护敏感信息 31 。此工作弥补了此前联邦图学习主要关注同构图的局限，为异构信息网络的联邦学习提供了借鉴。  
- 联邦图学习基准与平台方面，社区也在推进标准化：如OpenFedGraph项目提出统一的数据集和评测框架32；2022年KDD Explorations接收了联邦图学习的综述论文33，2025年又有更系统的综述发表在Neurocomputing34。这些综述总结了联邦图学习问题分类、主流技术和应用，以及现存局限和未来方向，为研究者提供了全面指引35 36。

综上，联邦图学习经历了初创探索（~2020）、方法繁荣（2021）到聚焦难题（2022-2023）的发展过程。早期工作奠定了horizontal/vertical两类基本范式，以及隐私保护的基本思路；随后大量改进方法涌现，从效率优化、自监督增强到自动化架构设计，丰富了技术储备；最近的研究开始深入应对联邦图特有的挑战，如数据和结构的高度异质、隐私攻击防御等，从而推动该领域逐渐走向成熟。

# 2. 安全问题：隐私保护、攻击与防御

由于联邦图学习涉及将敏感图数据留存在本地，在协同训练过程中必须高度关注数据隐私保护和潜在的攻击威胁。研究者在提高模型性能的同时，也在不断探索防范信息泄露和恶意攻击的方法。以下按时间顺序梳理联邦图学习安全相关研究的进展。

早期隐私保护措施（2020-2021）：首批联邦图学习工作已开始考虑基本的隐私机制。例如VFGNN（Chen等，2020/2022）在纵向场景中对服务器端输出应用了差分隐私扰动，防止服务器通过中间表示反推用户原始特征37。Wu等人在FedGNN/FedPerGNN中，则对每轮上传的本地梯度应用本地差分隐私，降低用户行为被推断的风险38。同时，为保护用户隐私，该方法提出引入虚假交互项（即随机采样一些未发生过的用户-物品交互）混淆真实数据20。这些机制在不大幅牺牲模型准确度的前提下，提高了联邦图推荐的隐私安全性21。可以看出，在联邦图学习兴起之初，研究就已认识到模型更新中可能泄露图结构和节点特征的风险3940。因此，将联邦学习中常用的差分隐私、防止过拟合的正则等手段移植过来成为首选方案。

攻击威胁识别与分析（2021-2023）：随着联邦图应用增多，对抗攻击的研究开始出现。Jinyuan Chen等人提出Graph-Fraudster（2022年IEEE TCSS），这是针对纵向联邦图场景的首个对抗攻击研究41。攻击者可以在只参与部分计算的情况下，通过精心设计的对抗样本干扰图模型，从而欺骗联邦学习过程。Graph-Fraudster揭示了纵向场景下GNN的脆弱性，加速了社区对模型鲁棒性的重视。此外，还有工作将已存在于集中式GNN中的攻击手段拓展到联邦场景。例如，Xu等研究了对联邦GNN的成员关系推断攻击，证明即使攻击者无法直接访问他人数据，也可依据模型更新推断某数据是否属于某客户端4243。与此同时，Zhang等在2023年初探讨了属性推断攻击在联邦图中的威胁44。这些研究表明，联邦学习并非天然安全，图结构的关系归纳偏置可能导致模型更新中泄露关于节点和边的敏感信息4546。特别是图数据的连通性使攻击者可利用邻居节点的影响推断未观测部分，从而衍生出比传统联邦学习更复杂的攻击形式4748。

系统化安全研究与防御（2024-至今）：联邦图学习的隐私与安全问题在近年受到系统性梳理和深入研究。例如，Ge等人在2024年发表了关于联邦图神经网络隐私研究的综述49，全面总结了联邦GNN面临的隐私泄露风险、已知攻击方法（如成员推断、属性推断、梯度重构等）以及相应防御策略。这标志着社区已充分认识到安全问题的紧迫性。针对防御，一方面，继续沿用差分隐私和安全多方计算等传统手段：如有研究提出在联邦图训练中引入本地差分隐私机制保护节点嵌入50；或采用安全聚合协议避免服务器在聚合时窥视单一客户端梯度51。另一方面，一些专门面向图数据的防御开始出现。例如，针对成员推断攻击的新威胁，Li等（2025年）提出了跨客户端成员推断攻击（CC-MIA）的防御设想，包括降低更新中的结构信息泄露以及设计鲁棒的GNN聚合规则52 46。此外，常规的对抗训练思想也被引入联邦图领域，以提升模型抵御对抗样本的能力。有工作通过在训练时注入噪声或扰动图结构来增强模型鲁棒性42。总的来说，近年防御研究正从简单的噪声添加，走向结合联邦图特性的综合方案，例如：兼顾精度与隐私的多目标优化，利用图结构同态加密保护邻居关系，或实时检测恶意客户端的异常更新。这些方向都在积极探索中。

需要强调的是，联邦图学习的安全问题具有更高复杂性：除了传统联邦学习中的数据分布、模型更新泄露外，图数据独有的结构信息可能通过梯度、嵌入的细微变化而泄漏53。因此，未来研究需要针对图结构的隐私保护提出更有针对性的解决方案。例如开发结构化噪声添加技术，在保证图模式隐私的同时尽量保持模型性能；设计联邦图特有的攻击检测算法，实时发现异常的模型更新行为等。

# 3. 异构图场景下的联邦学习

现实世界中的图往往是异构信息网络，包含多种类型的节点和边（如社交网络中用户和帖子、知识图谱中实体和关系）。此外，在联邦场景，不同客户端的图数据模式可能存在跨域差异，如节点ID不共享、类型分布各异等。这给联邦图学习带来额外挑战：如何在异构图和跨域场景下有效对齐和学习？近年一些研究专门面向异构图联邦学习展开，下面按照时间顺序介绍代表性进展。

知识图谱和多关系数据的联邦嵌入（2021）：异构图的典型代表是知识图谱，包含多种关系类型。Min Chen等人在2021年提出了FedE: Embedding Knowledge Graphs in Federated Setting 23。这是首个将联邦学习用于知识图谱表示的框架。各参与方训练本地的知识图谱嵌入模型，例如基于TransE等技术54，服务器定期聚合更新。由于不同知识库可能有重叠实体，FedE重点解决对齐问题：通过各方共享一部分嵌入或利用公共标识，将同一实体的分布式表示对齐，从而实现跨库知识融合。该工作展示了在不集中数据的情况下，多方如何协作学习知识图谱 embedding，为复杂异构网络的联邦学习提供了范例。

纵向异构联邦图：在异构图场景下，不仅节点种类多样，而且不同客户端可能持有互补的属性或关系。VFGNN（Chen等，2022）虽主要针对纵向数据拆分，但其方法实际上适用于异构场景：通过将特征、边等计算分离给不同方处理，再在服务器上融合5。这解决了异构数据在不同方之间无法直接合并的问题。例如，在推荐系统中，一个客户端掌握用户特征，另一个掌握用户关系网络，二者结合才能完整构建异构图模型。VFGNN提供的分离计算思想，为此类跨源异构数据的联合建模奠定基础。

异构图神经网络的联邦训练（2023）：Bo Yan等人在2023年的工作FedHGNN（联邦异构图神经网络）进一步面向带有多类型节点/边的图设计联邦学习方案55。该框架在本地采用异构图神经网络（如基于元路径或关系注意力的模型）对不同类型节点分别建模，然后通过联邦平均或更复杂的聚合策略融合全局模型。由于异构图的本地模型参数通常按节点/边类型划分，FedHGNN探讨了分类型参数聚合的方法：例如只聚合共享的部分（针对各客户端都有的节点类型），而对客户端特有的类型执行个性化本地训练。这种做法可在一定程度上解决多样化模式的对齐：各客户端在公共模式上协同，在私有模式上保留差异，从而避免不兼容的数据强行融合带来的性能下降。FedHGNN验证了该思路在隐私保护推荐场景的有效性55。类似地，学者们也提出针对异构信息网络的联邦分布式表示学习、联邦链接预测等方法，使得不同结构、语义的图数据也能参与联邦训练。

跨域对齐与联邦迁移学习：当联邦中的不同客户端图属于不同领域（domain）时，往往存在跨域信息对齐的问题。例如在跨社交平台的联邦学习中，不同平台的用户节点虽然代表相同实体，但ID和关系并不共享；再如跨领域推荐中，不同行业的用户行为图需要部分知识迁移。为此，近期一些研究开始探索联邦图的跨域迁移。Ziqi Yang等在NeurlPS 2024提出FedGCDR: Federated Graph Learning for Cross-Domain Recommendation，通过引入特征映射和差分隐私的知识提取模块，将源域的有用知识安全地迁移到目标域56 57。具体来说，FedGCDR用联邦训练的图注意力网络提取源域用户嵌入，然后通过映射函数对齐到目标域的特征空间，并利用差分隐私确保映射过程不泄露源域隐私58。同时，设计知识激活模块过滤掉可能不适用的源域知识，避免负迁移59。这种方法展现了跨域联邦图模型的潜力，在多源跨域推荐实验中显著优于仅靠目标域训练的模型60。这类研究表明，通过巧妙的对齐和知识蒸馏，联邦图学习可以突破单域限制，实现跨领域协同，这是异构图联邦学习的重要发展方向。

总的来看，异构图场景为联邦学习提出了更高要求：既要处理多样的节点和关系类型，又要解决跨客户端实体对齐和知识转移难题。现有工作提供了一些可行策略，如共享部分嵌入对齐实体、分类型聚合、联邦迁移模块等。然而，全面有效的解决方案仍在探索中。例如，对于没有公共实体ID的异构图，可考虑引入隐私保护的实体对齐协议（如基于同态加密的ID匹配）；对于跨域差异巨大的情况，需要新的联邦多任务学习范式，在保证模型泛化性的同时适应各域特殊性。这些都属于未来值得深耕的方向。

# 4. 联邦图学习在推荐系统中的进展

推荐系统是联邦图学习的重要应用场景之一。在推荐任务中，用户-物品交互可以自然建模为二部图或更复杂的关系图，利用GNN能捕获高阶关联提升推荐效果 61 62 。但由于用户行为数据高度敏感，如何在保护隐私的前提下协同训练推荐模型成为研究热点 63 。联邦图学习为此提供了新途径。下面按照时间顺序介绍联邦图学习在推荐系统领域的代表性工作及其演化。

- 联邦社交推荐 with GNN（2020-2021）：早期一些研究将联邦学习用于社交推荐。例如Wu等（2020）探索了联邦矩阵分解和联邦图卷积结合，用于社交网络的好友推荐。然而这些方法多采用简单聚合策略，未充分利用图结构信息。真正将GNN引入联邦推荐并产生较大影响的是前述的FedGNN/FedPerGNN（Wuet al., 2021/2022）1864。它在每个客户端构建用户的本地用户-物品二部图，利

用GraphSAGE或GCN捕捉交互的高阶结构19。通过联邦框架聚合全局模型参数，再分发回本地迭代。FedGNN率先证明了联邦GNN在推荐任务中的有效性：在多个新闻、商品推荐数据集上，其Top-K推荐性能接近甚至优于集中式GNN模型，同时显著减少了隐私风险65。这一成果发表在Nature Communications上，引起广泛关注，被视作联邦图推荐的里程碑。

- 垂直联邦推荐：在推荐场景中，常出现数据垂直分割：如一家平台有用户行为，另一家有商品属性。为此Chen等提出P4GCN：Privacy-Preserving Personalization via Federated Graph Convolutional Networks（2021），采用双方计算模式：一方计算用户侧GNN，一方计算物品侧GNN，中间通过加密协议交换信息 66。这种两方联邦图推荐实现在不共享原始行为数据的情况下，融合了不同持有者的数据，为跨企业/部门的联合推荐提供了范例。P4GCN等工作丰富了联邦推荐的实现形式。

·通信与效率优化（2021-2022）：由于推荐系统用户数庞大、模型复杂，通信效率成为联邦推荐落地的关键。为降低通信频率，Haiyan Wang等提出CFedGR: Cluster-Enhanced Federated Graph Recommendation (2022年) 67 68 。CFedGR在服务器执行用户嵌入聚类，将兴趣相近的用户分组，挑选代表用户进行模型更新 69 70 。这样每轮只需部分用户参与，大幅减少通信和计算成本。此外CFedGR限制每个用户只与top-k相似邻居交换信息，进一步压缩传输量 71 。实验显示即便减少通信，聚类策略仍能有效引入高阶协同信号，推荐效果优于各客户端孤立训练 72 68 。类似地，FedGraph采用的智能采样子图策略 73 、以及一些稀疏化梯度传输的方法，也能提升联邦推荐的可扩展性。这些研究指向了一个趋势：通过分层聚合和用户调度来折衷精度与效率，使联邦图推荐有望应用于大规模实际系统。

- 联邦图推荐的新方向（2023-2025）：近期，联邦图推荐研究开始向更复杂和实际的问题延伸。一方面，会话推荐（session-based recommendation）等场景引入了序列图的联邦学习。Lou等人在2025年报道了Efficient Federated Graph Aggregation for Privacy-Preserving GNN-based Session Recommendation（发表于Nature Communications）74。他们桥接了联邦学习与会话图神经网络的实践鸿沟，通过自适应的聚合策略显著提升了会话推荐的性能和效率（具体细节因未完全公开，这里不展开）。这是联邦图学习应用到序列推荐的新尝试。另一方面，跨域推荐结合联邦图学习也备受关注（如前述FedGCDR，NeurIPS 2024），旨在让多个领域的推荐系统共享知识而不违反隐私7556。此外，个性化联邦推荐也是热点，即根据客户端的特殊偏好微调全局模型，例如利用联邦元学习让每个用户获得专属的轻量个性化模型。总之，在推荐系统这个极具价值的应用领域，联邦图学习的研究正逐步深入，不仅关注精度提升，也注重保护隐私、提高效率和适应复杂场景。预计未来会有更多线上大规模实验和工业应用报告，推动该方向迈向成熟。

# 5. 挑战与未来趋势

尽管联邦图学习取得了诸多进展，但仍有诸多挑战亟待解决，未来研究需要在以下方面加大力度：

- 数据和结构异质性挑战：联邦图学习面临的最大挑战之一是不同客户端数据的高度异质性，包括节点特征分布差异、图拓扑结构差异、标签空间不一致等 176。现有方法（如元学习、自监督、结构共享等）只能部分缓解non-IID问题，但难以根本消除影响。未来需要研发更健壮的联邦优化算法，能够自适应地平衡各客户端贡献，或在服务器端引入图结构先验指导聚合 28。同时，可探索个性化联邦图学习，为不同客户端量身定制模型（例如采用参数分层、本地微调等）以适应其异质数据。  
- 隐私保护与安全对抗：随着攻击手段的发展，联邦图模型的隐私泄露面将更复杂。例如，攻击者可能通过全局模型推理出图中的隐蔽关系或社区结构。如何在保证模型效果的同时，严格限制信息泄露是长期课题。未来可能需要结合多重技术：差分隐私、同态加密、安全多方计算、可信执行环境等，共同构建联邦图的安全盾。针对新出现的攻击（如成员归属攻击45），需要设计专门的检测和防御机制，例如监控异常梯度更新以识别恶意客户端。总体而言，隐私-精度权衡将在更细粒度上考量，定制化的隐私保护策略或将应运而生。

- 效率与可扩展性：大规模图数据的联邦学习在通信、存储和计算上都面临瓶颈。如果每轮训练都需交换完整模型参数，对于包含数百万节点的大图将难以承受。未来研究应继续优化通信协议（如模型压缩、梯度剪枝）、客户端调度（如分批训练、异步更新）等。特别是在移动设备上的联邦图应用，需要考虑终端算力和带宽受限的问题，可探索高效模型结构（如轻量GNN、剪枝蒸馏）和高效训练算法。同时，开发联邦图学习系统工具包也是趋势，如统一的开源框架，支持从分布式训练、加密通信到结果评估的全流程，加速研究到应用的转化。  
- 算法与理论突破：当前联邦图学习很多方法仍以经验为主，理论分析相对滞后。例如，联邦图神经网络的收敛性和泛化性能缺乏严格证明，non-IID数据下的最优聚合策略也不明晰。未来需要发展相应的理论框架，回答如“图结构对联邦优化的影响机制”、“隐私保护机制对模型收敛的代价”等问题。这将有助于指导算法设计，避免纯粹试错。算法层面，值得关注的新思路包括：将图预训练模型（如预训练GNN）引入联邦范式，实现跨任务知识共享；融合自适应优化和联邦控制变量技术，减轻客户端漂移；以及联邦强化学习在图决策问题（如路径推荐）中的应用等等。  
- 丰富应用场景：联邦图学习的应用前景广阔，未来可能拓展到更多领域并面临新的课题。例如，在医疗保健领域，不同医院的患者关系图可通过联邦学习共享知识，但需满足严格法规和高可靠性要求36。在金融风控中，多机构的交易网络联邦学习可以提升诈骗检测，但需要解决跨机构标识匹配难题。在物联网中，传感器网络生成的大规模动态图如何进行联邦学习也是待解问题。每个新场景都会引发新的研究需求，如动态联邦图学习（处理随时间演化的图）以及联邦图上的解释性AI（帮助不同行参与方理解模型决策）。这些方向都有待深入探索。

综上，联邦图学习作为一个新兴交叉领域，已经构建了从基础框架、安全机制到具体应用的初步版图。未来的研究将围绕更强的鲁棒性、更高的安全性、更大的规模和更多元的应用展开。在技术不断演进和隐私合规需求日益增长的双重驱动下，联邦图学习有望成为支撑多方数据协同分析的重要范式。可以预见，随着上述挑战的逐步克服，联邦图学习将在学术研究和工业实践中发挥日益关键的作用。我们有理由期待，一个安全、高效、智能的联邦图学习生态将在不久的将来成熟完善。
